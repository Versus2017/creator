# 模型与密钥配置

Creator 的 API Key、对话模型、生图模型均通过 **`creator-api/creator/local_config.env`** 管理（该文件已被 Git 忽略，不会提交）。

```bash
cd creator-api
cp creator/local_config.env.example creator/local_config.env
# 编辑 local_config.env，填入真实密钥
```

修改后需**重启 API 服务**（`make run`）和 **Huey worker**（`make worker`）生效。

---

## 一、配置文件位置

| 文件 | 作用 |
|------|------|
| `creator/local_config.env` | **本地私密配置**（密钥、模型选型、URL 覆盖） |
| `creator/local_config.env.example` | 示例模板（可提交 Git） |
| `creator/config.py` | 默认值与字段定义（密钥默认为空） |

`local_config.env` 中的变量会**覆盖** `config.py` 里的同名项。

---

## 二、对话模型配置（`CHAT_*`）

Creator 在不同场景会使用不同的「模型类型」，通过 `model_type` 路由到具体 AI 客户端。

### 模型类型一览

| `model_type` | 说明 | 需配置的 Key |
|--------------|------|----------------|
| `openai_gpt` | OpenAI 兼容对话（默认推荐） | `OPENAI_GPT_API_KEY` |
| `deepseek` | DeepSeek 对话 | `DEEPSEEK_API_KEY` |
| `ali_chat` | 通义千问（DashScope 兼容） | `ALY_MODEL_API_KEY` |
| `doubao_vision` | 豆包视觉模型 | `DOUBAO_API_KEY` |
| `doubao_16` | 豆包 1.6 深度思考 | `DOUBAO_16_API_KEY` |

### 场景与配置项

| 配置项 | 作用场景 | 默认值 |
|--------|----------|--------|
| `CHAT_CONVERSATION_MODEL_TYPE` | Web 创作对话、脚本生成、研究对话 | `openai_gpt` |
| `CHAT_CODEX_MODEL_TYPE` | Codex / MCP 创建的首条创作对话 | `openai_gpt` |
| `CHAT_SCRIPT_MEDIA_PLAN_MODEL_TYPE` | 脚本 AI 生图前的「视觉规划」分析 | `openai_gpt` |
| `CHAT_ALLOW_REQUEST_MODEL_OVERRIDE` | 为 `True` 时，前端可传 `model_type` 覆盖创作对话模型 | `False` |

### 示例

```env
# 创作对话走 OpenAI 兼容 GPT
CHAT_CONVERSATION_MODEL_TYPE=openai_gpt
CHAT_CODEX_MODEL_TYPE=openai_gpt
CHAT_SCRIPT_MEDIA_PLAN_MODEL_TYPE=openai_gpt
CHAT_ALLOW_REQUEST_MODEL_OVERRIDE=False
```

若希望创作对话改用 DeepSeek：

```env
CHAT_CONVERSATION_MODEL_TYPE=deepseek
DEEPSEEK_API_KEY=sk-xxx
```

## 三、OpenAI 兼容对话（`openai_gpt`）

用于创作对话、脚本提取、Codex 首条回复等（当 `CHAT_*_MODEL_TYPE=openai_gpt` 时）。

| 配置项 | 说明 | 默认 |
|--------|------|------|
| `OPENAI_GPT_API_KEY` | API Key | 空 |
| `OPENAI_GPT_API_BASE` | 兼容接口 Base URL | `https://api.laozhang.ai/v1` |
| `OPENAI_GPT_MODEL_NAME` | 模型名称 | `gpt-5.5` |
| `GPT_PROXY` | 可选 HTTP 代理 | 空 |

```env
OPENAI_GPT_API_KEY=sk-xxx
OPENAI_GPT_API_BASE=https://api.laozhang.ai/v1
OPENAI_GPT_MODEL_NAME=gpt-5.5
GPT_PROXY=
```

也可改用 OpenAI 官方或其他兼容网关，只需修改 `OPENAI_GPT_API_BASE` 与 Key。

---

## 四、脚本 AI 生图（`gpt-image-2`）

封面与段落素材图生成，**独立于**对话模型配置。

| 配置项 | 说明 | 默认 |
|--------|------|------|
| `GPT_IMAGE_2_API_KEY` | 生图 API Key（可与对话 Key 相同） | 空 |
| `GPT_IMAGE_2_API_URL` | 生图接口 Base URL | `https://api.laozhang.ai/v1` |
| `GPT_IMAGE_2_MODEL_NAME` | 模型名 | `gpt-image-2` |
| `GPT_IMAGE_2_ROUTE` | 线路标识 | `enterprise` |
| `GPT_IMAGE_2_DEFAULT_QUALITY` | 默认画质 | `high` |
| `GPT_IMAGE_2_REQUEST_TIMEOUT` | 请求超时（秒） | `240` |

```env
GPT_IMAGE_2_API_KEY=sk-xxx
GPT_IMAGE_2_API_URL=https://api.laozhang.ai/v1
```

封面默认尺寸等在 `config.py` 中定义（如竖版 `2048x2048`、横版 `2048x1152`），一般无需在 `local_config.env` 修改。

## 五、其他 AI 服务 Key

按需填写，仅在使用对应 `model_type` 时需要。

| 配置项 | 用途 |
|--------|------|
| `DEEPSEEK_API_KEY` | DeepSeek 对话 |
| `ALY_MODEL_API_KEY` | 通义 / DashScope（`ali_chat`） |
| `DOUBAO_API_KEY` | 豆包视觉（`doubao_vision`） |
| `DOUBAO_16_API_KEY` | 豆包 1.6（`doubao_16`） |
| `GPT_TOKEN` / `ZHIPU_API_KEY` | 历史或其他模块（按实际使用填写） |

阿里云短信 / OCR（若启用）：

```env
ALY_ACCESS_KEY_ID=
ALY_ACCESS_KEY_SECRET=
OCR_ALY_ACCESS_KEY_ID=
OCR_ALY_ACCESS_KEY_SECRET=
```

## 六、语音转写（Whisper）

| 配置项 | 说明 | 默认 |
|--------|------|------|
| `WHISPER_USE_LOCAL` | `True` 用本地 faster-whisper；`False` 走 OpenAI API | `True` |
| `WHISPER_MODEL_NAME` | 本地模型：`tiny` / `base` / `small` / `medium` / `large` | `medium` |
| `WHISPER_DEVICE` | `cpu` 或 `cuda` | `cpu` |
| `WHISPER_COMPUTE_TYPE` | `int8` / `float16` / `float32` | `int8` |
| `OPENAI_API_KEY` | 云端 Whisper 时使用 | 空 |

```env
WHISPER_USE_LOCAL=True
WHISPER_MODEL_NAME=small
WHISPER_DEVICE=cpu
WHISPER_COMPUTE_TYPE=int8
```

## 七、Codex / MCP 鉴权

与 [Creator MCP 配置说明](../../creator-mcp/README.md) 配合使用。

| 配置项 | 说明 |
|--------|------|
| `CREATOR_CODEX_TOKEN` | Codex 请求头 `X-Codex-Token`，需与 MCP 环境变量一致 |
| `CREATOR_CODEX_USER_ID` | 可选，指定代操作的 Creator 用户 ID |

```env
CREATOR_CODEX_TOKEN=codex-local
CREATOR_CODEX_USER_ID=1
```

## 八、对话上下文限制（可选）

以下在 `config.py` 有默认值，一般无需改；若需调大上下文可在 `local_config.env` 覆盖：

| 配置项 | 含义 | 默认 |
|--------|------|------|
| `CHAT_MAX_HISTORY_MESSAGES` | 对话历史条数上限 | `36` |
| `CHAT_MAX_MESSAGE_CHARS` | 单条消息字符上限 | `6000` |
| `CHAT_MAX_REFERENCED_SCRIPTS` | 最多引用历史脚本篇数 | `3` |
| `CHAT_MAX_SCRIPT_REF_CHARS` | 单篇引用脚本字符上限 | `4000` |
| `CHAT_MAX_SCRIPT_REF_TOTAL_CHARS` | 引用脚本总字符上限 | `10000` |

## 九、最小可运行示例

仅体验 Web 创作 + 本地 Whisper（对话用 DeepSeek）：

```env
SECRET_KEY=change-me-local
AUTHJWT_SECRET_KEY=change-me-local
EXTERNAL_URL=http://127.0.0.1:8000

DEEPSEEK_API_KEY=sk-xxx
CHAT_CONVERSATION_MODEL_TYPE=deepseek

WHISPER_USE_LOCAL=True
WHISPER_MODEL_NAME=small
```

完整功能（OpenAI 兼容对话 + 生图 + MCP）：

```env
SECRET_KEY=change-me-local
AUTHJWT_SECRET_KEY=change-me-local
EXTERNAL_URL=http://127.0.0.1:8000

OPENAI_GPT_API_KEY=sk-xxx
GPT_IMAGE_2_API_KEY=sk-xxx
CHAT_CONVERSATION_MODEL_TYPE=openai_gpt
CHAT_CODEX_MODEL_TYPE=openai_gpt
CHAT_SCRIPT_MEDIA_PLAN_MODEL_TYPE=openai_gpt

CREATOR_CODEX_TOKEN=codex-local
CREATOR_CODEX_USER_ID=1

WHISPER_USE_LOCAL=True
WHISPER_MODEL_NAME=small
```
